# 📚 2026-1H 디지털 논리 회로 (Digital Logic Circuits)

본 저장소는 숭실대학교 디지털 논리 회로 강의의 **실습용 데이터**를 제공합니다.

### 📢 강의 안내
* **과제 제출**: 모든 과제는 학교 **LMS(강의지원시스템)**에 제출해야 합니다.
* **데이터 활용**: 주차별 폴더에 업로드된 Data를 다운로드하여 실습에 활용하세요.
* 본 폴더의 실습 데이터(.zip)는 보안을 위해 비밀번호가 설정되어 있습니다.
비밀번호: 수업 시간에 공지된 암호를 입력하세요. (또는 학교 LMS 공지사항 참조)
